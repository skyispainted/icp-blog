# ICP备案技术博客设计文档

## 项目概述

创建一个单网页技术博客，用于通过ICP备案审查。博客内容为AI/Claude技术分享，采用仿WordPress经典博客风格。

---

## 技术约束

- **单HTML文件**：所有内容集中在index.html
- **外部资源**：JS和图片全部从外部CDN引入，本地无资源文件
- **国内CDN**：使用BootCDN确保备案友好
- **备案信息**：底部必须展示ICP备案号（预留位置）

---

## 页面结构

```
+------------------------------------------+
|  HEADER: 博客标题 + 副标题描述            |
+------------------------------------------+
|  NAV: 首页 | 技术分享 | 关于我            |
+------------------------------------------+
|  SIDEBAR (25%)    |   MAIN CONTENT (75%) |
|  - 分类列表       |   - 文章列表          |
|  - 标签云         |   (3-5篇文章摘要)     |
|  - 友情链接       |                      |
+------------------------------------------+
|  FOOTER: 版权 | 备案号 | 联系方式         |
+------------------------------------------+
```

---

## 外部资源清单

### CSS（BootCDN）
- Bootstrap 4.5.3 CSS: `https://cdn.bootcdn.net/ajax/libs/twitter-bootstrap/4.5.3/css/bootstrap.min.css`

### JS（BootCDN，可选）
- Bootstrap Bundle: `https://cdn.bootcdn.net/ajax/libs/twitter-bootstrap/4.5.3/js/bootstrap.bundle.min.js`

### 图片
- 不使用外部图片，采用纯色背景或SVG图标
- 避免图片加载失败影响备案审查

## 安全合规要点

1. **内容合规**：文章内容不涉及敏感话题，纯技术分享
2. **链接合规**：不包含外链到未备案网站
3. **资源合规**：使用国内CDN，避免被墙资源
4. **结构规范**：页面结构清晰，无隐藏内容
5. **备案号预留**：底部预留备案号位置，格式规范
